    I had the same problems (trying to write even larger CSV files) and decided finally againt using CSV files.

    I would recommend you to use SQLite as it is much faster than dealing with CSV files:

    1) Just install SQLite: http://www.sqlite.org/download.html

    2) Create a new database using R and save your data.table in this database 

    require("RSQLite")
# Set up database    
drv <- dbDriver("SQLite")
tfile <- "test.db"
con <- dbConnect(drv, dbname = test.db)
# Load example data
data(USArrests)
# Write data "USArrests" in table "USArrests" in database "test.db"    
dbWriteTable(con, "arrests", USArrests)
dbDisconnect(con)

    For further informtion, see http://cran.r-project.org/web/packages/RSQLite/RSQLite.pdf

    You can also use a software like http://sqliteadmin.orbmu2k.de/ to access the database and export the database to CSV etc.
