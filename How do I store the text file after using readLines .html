    saveRDS is your friend. I've built a whole pipeline around it.

    BTW, I've found that if you're lucky enough to have a relatively nice computer (e.g., something with a fast SSD and 8+ GB RAM) there's no reason to read the lines in 1000 at a time -- I just did readLines on all three files and appended them as a data frame:

    types <- c('twitter', 'blogs', 'news')

# readSourceFile calls readLines on 'data/source/en_US/en_US.(type).txt' and returns a dataframe
# with $source=(type) and $text=(readLines result with bad utf-8 stripped out using iconv)
rawData <- do.call('rbind', lapply(types, readSourceFile))

# after some consistency checks...
saveRDS(rawData, 'data/raw-data.RDS')
